# Knowledge Base

This is an advanced documentation section. Usually you only need to read it if you want to understand the engine internals, or if you want to do some advanced modding with IdaJS. It is not required for basic modding, but it can be useful for better understanding of how the engine works.

The knowledge articles below cover in-depth some of the technical aspects about the LBA2 engine internals, including those, relevant to IdaJS modding.

- [Decors](Decors.md) - Exterior 3D fragments, that can be controlled by game variables
- [Fragments](Fragments.md) - Isometric grid patches for indoor maps, that can be controlled by Life script commands
- [SaveLoad](SaveLoad.md) - Scene loading flow, save/load system, and persisted game state
